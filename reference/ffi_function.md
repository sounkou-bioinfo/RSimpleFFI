# Create a reusable FFI function wrapper

Create a reusable FFI function wrapper

## Usage

``` r
ffi_function(name, return_type, ..., library = NULL)
```

## Arguments

- name:

  Character name of the function

- return_type:

  FFIType for return value

- ...:

  FFIType objects for arguments

- library:

  Character name of library (optional)
