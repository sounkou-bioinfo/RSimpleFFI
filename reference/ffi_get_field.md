# Get field value from FFI structure

Get field value from FFI structure

## Usage

``` r
ffi_get_field(ptr, field, struct_type, ...)
```

## Arguments

- ptr:

  External pointer to structure

- field:

  Character field name or integer field index

- struct_type:

  StructType object

- ...:

  Additional arguments (not used)

## Value

Field value
