# FFI Call Interface (CIF)

FFI Call Interface (CIF)

## Usage

``` r
CIF(return_type = FFIType(), arg_types = list(), ref = NULL)
```

## Arguments

- return_type:

  FFIType for return value

- arg_types:

  List of FFIType objects for arguments

- ref:

  External pointer to ffi_cif

## Value

An CIF object
