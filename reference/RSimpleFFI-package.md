# RSimpleFFI: Simple Foreign Function Interface using 'S7' and 'libffi'

Simple Foreign Function Interface for 'R' using 'libffi' and 'S7'
classes. Supports calling 'C' functions with some type conversion.
Includes standard 'C' types (int8, int16, int32, int64, uint variants),
platform types (size_t, bool), and floating point types.
